## 📞 问题排查

### YouTube下载失败
- 检查网络连接
- 更新yt-dlp: `pip3 install -U yt-dlp`

### 字幕提取失败
- YouTube自动生成字幕可能有延迟
- 尝试使用 `--write-sub` 而非 `--write-auto-sub`

### 视频无法播放
- 使用VLC播放器（支持SRT字幕加载）
- 或使用QuickTime + 单独加载字幕
