# 4天增长运营大师课-demo

一个自用的增长与运营知识学习网站。每天 1 小时、共 4 天，系统掌握增长核心框架。

**在线访问：** https://siafan0625.github.io/growth-demo/

## 这是什么

这是一个将公开、免费、权威的增长与运营知识整理成结构化课程的 Web 应用。内容涵盖：

| Day | 主题 | 来源 |
|-----|------|------|
| Day 1 | 用户增长万能公式 | AARRR 模型 + 八角行为分析法 |
| Day 2 | 多邻国增长策略 | Duolingo 增长案例拆解 |
| Day 3 | PMF 层级框架（上） | Product-Market Fit 方法论 |
| Day 4 | PMF 层级框架（下） | PMF 进阶与实践 |

适合增长、运营方向的行业新人学习使用。

## 如何使用

1. 打开 [在线地址](https://siafan0625.github.io/growth-demo/)，进入首页
2. 选择任意一天开始学习，每天包含 3 个单元（导读与框架 / 精读 / 实践与总结）
3. 每个单元支持 **速览**（快速掌握要点）和 **详读**（深入阅读原文）两种模式
4. 学习进度自动保存在浏览器本地，下次打开会继续上次的位置
